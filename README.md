DonutAuctions

Quality of life Mod for donutsmp.net that allows you to see the lowest Price an Item sells for on the Auction House, just by hovering over it. The "value" is calculated by taking the cheapest item on the Auction House, that matches your item regarding

    material
    enchantments
    (soon) armor trims

Setup (Important!!!)

This Mod uses the DonutSMP API in Order to query active Auctions. To do this, the Mod needs your API-Key. By Running the /api on Donutsmp.net, the Mod automatically copies your API-Key and stores it. This means you only need to run this Command ONCE, when first Using the Mod. The Mod will not use your API key for anything but querying Auctions
